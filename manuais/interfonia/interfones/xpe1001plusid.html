CONFIGURAÇÕES

⚠️O XPE precisa estar conectado a uma LINHA de ramal, seja PABX ou ATA.

Com um telefone, disque para o ramal do XPE 1001 Plus ID, que atenderá automaticamente e confirmará com dois bipes longos, em seguida, siga os passos:

  1. Tecla Única
    • #4(NÚMERO A SER PROGAMADO)##
      • EX: # 4 1234 ##
  2. Tempo de conversação
    • # + 6 + T + # #
      • » Onde T é o tempo de conversação de 0 a 9, múltiplo de 30 segundos (padrão de fábrica: 3 [90 segundos]).
  3. Programação do volume de recepção (alto-falante)
    • # + 792 + V + # #
    • V = 1: configura o volume no nível baixo.
    • V = 2: configura o volume no nível médio (padrão de fábrica).
    • V = 3: configura o volume no nível alto.
  4. Configurar 2° dígito para acionamento das saídas
    • # + 75 + S + D + # #
      • Onde o S é a saída 1 ou 2 e D é o novo Dígito.
